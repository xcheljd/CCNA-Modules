## How to Update Screenshots

If you need to retake screenshots:

1. Launch the app: `npm start`
2. Navigate to the view you want to capture
3. Use a nice theme (Nord, Catppuccin, or Rose Pine recommended)
4. Take screenshot (Cmd+Shift+3 on macOS, or Cmd+Shift+4 for selection)
5. Save to this directory with descriptive filename
6. Update README.md with new image reference if filename changes

## Guidelines for Good Screenshots

- Use a clean, uncluttered view
- Show real data (not empty states when possible)
- Use attractive themes
- Ensure text is readable
- Capture at standard resolution (1920x1080 recommended)
