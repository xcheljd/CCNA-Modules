Option 2: Screenshot Method

  1. Open this page in browser (zoom to 100%)
  2. Take screenshot of the 512x512 icon (Cmd+Shift+4 on Mac)
  3. Crop to exactly 512x512 pixels
  4. Save as PNG and use online converter

Option 3: Install ImageMagick

  1. Install: brew install imagemagick (macOS)
  2. Convert SVG: convert assets/icon.svg -resize 512x512 build/icon.png
  3. Create ICO: convert assets/icon.svg -resize 256x256 build/icon.ico
  4. Follow build/README.md for macOS .icns creation
